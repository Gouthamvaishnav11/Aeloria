Introduction

Welcome to Aeloria — a lightweight deployment platform that connects to your GitHub repositories, runs tests, and deploys applications automatically.

This documentation covers everything from getting started to advanced configuration and API usage.

Quickstart

Get your first app live in minutes with three simple steps:

  1. Connect your GitHub repository.
  2. Configure build & environment (auto-detected for most projects).
  3. Deploy and access your app at a custom domain.
git clone https://github.com/you/your-app.git
cd your-app
# push to GitHub, then link the repo in Aeloria
# Aeloria will detect the framework and create a pipeline

Configuration

Aeloria supports an optional aeloria.yaml file for custom builds.

# aeloria.yaml
build:
  command: "npm run build"
  output: "build/"
env:
  NODE_ENV: production
  API_URL: "https://api.example.com"

CLI

Use the Aeloria CLI to manage deployments locally.

# install
npm i -g @aeloria/cli

# login
aeloria login

# deploy current repo
aeloria deploy --prod

API Reference

Aeloria exposes a REST API for programmatic deployments.

Trigger a deployment

POST /v1/deploy
Authorization: Bearer <API_KEY>
Content-Type: application/json

{
  "repo": "github.com/you/your-app",
  "branch": "main"
}

Successful response returns a deployment_id you can use to poll status.

FAQ

How long does a deployment take?

Typical builds finish in under 2 minutes for small apps. Larger builds depend on project complexity.

Can I use a custom domain?

Yes — add your domain in the dashboard and follow the DNS setup instructions we provide.
